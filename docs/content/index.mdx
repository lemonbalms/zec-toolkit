# ZEC Toolkit

Claude Code를 위한 개발 오케스트레이션 플러그인

---

## 시작하기

ZEC Toolkit은 [MoAI-ADK](https://github.com/modu-ai/moai-adk)를 기반으로 만든 Claude Code 플러그인입니다.
에이전트 오케스트레이션, 스킬 시스템, 품질 게이트 등 실제로 자주 쓰는 기능만 추려서 가볍게 재구성했습니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| **ZEC 오케스트레이터** | 27개 전문 에이전트를 자동 라우팅하는 전략 지휘관 |
| **Harness 스킬 9종** | FastAPI, pytest, Next.js, React 성능, Mermaid 등 |
| **SPEC 기반 개발** | EARS 형식 요구사항 → 체계적 구현 파이프라인 |
| **TRUST 5 품질** | Tested, Readable, Unified, Secured, Trackable |
| **Rules & Templates** | 프로젝트 간 일관된 개발 환경 설정 |
