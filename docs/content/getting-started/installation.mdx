# 설치

ZEC Toolkit을 Claude Code에 설치하는 방법을 안내합니다.

import { Callout, Steps, Tabs } from 'nextra/components'

## 사전 요구사항

| 항목 | 요구사항 | 확인 방법 |
|------|----------|-----------|
| **Claude Code** | 최신 버전 | `claude --version` |
| **Git** | 2.x 이상 | `git --version` |
| **Node.js** | 18 이상 (선택) | `node --version` |

<Callout type="warning">
Claude Code가 설치되어 있지 않다면 먼저 [Claude Code 공식 문서](https://docs.anthropic.com/en/docs/claude-code)를 참고하여 설치하세요.
</Callout>

### 마켓플레이스 등록

Claude Code 내에서 마켓플레이스를 등록합니다:

```
/plugin marketplace add lemonbalms/zec-toolkit
```

### 플러그인 설치

```bash
claude plugin install zec-toolkit@lemonbalms-zec-toolkit
```

설치 범위를 지정할 수 있습니다:

```bash
# 모든 프로젝트에서 사용 (기본값)
claude plugin install zec-toolkit@lemonbalms-zec-toolkit --scope user

# 팀 공유 (.claude/settings.json에 기록)
claude plugin install zec-toolkit@lemonbalms-zec-toolkit --scope project

# 현재 저장소만
claude plugin install zec-toolkit@lemonbalms-zec-toolkit --scope local
```

### 설치 확인

Claude Code에서 `/zec` 명령어가 인식되는지 확인합니다:

```
> /zec
```

ZEC 오케스트레이터가 응답하면 설치가 완료된 것입니다.

</Steps>

Claude Code에서 `/plugin` 명령어를 입력하면 인터랙티브 UI가 열립니다:

- **Discover** — 마켓플레이스에서 플러그인 검색 및 설치
- **Installed** — 설치된 플러그인 관리 (활성화/비활성화/제거)
- **Marketplaces** — 마켓플레이스 추가/제거
- **Errors** — 플러그인 로딩 오류 확인

### 프로젝트 초기 설정

설치 후 프로젝트 디렉토리에서 초기 설정을 진행합니다. 자세한 내용은 [초기 설정](/getting-started/init-wizard) 문서를 참고하세요.

## 포함되는 항목

설치 시 다음이 자동으로 Claude Code에 등록됩니다:

| 항목 | 수량 | 설명 |
|------|------|------|
| **Skills** | 9개 | ZEC 오케스트레이터 + Harness 스킬 8종 |
| **Rules** | 6개 | 핵심 원칙, 코딩 표준, 워크플로우 규칙 |
| **Templates** | 15개 | `.zec/config/` YAML 설정 파일 |

<Callout type="info">
Rules는 플러그인 설치 시 자동으로 활성화됩니다. Skills는 해당 작업 컨텍스트에서 자동 트리거되거나 `/zec` 명령으로 수동 실행할 수 있습니다.
</Callout>

## 다른 플러그인과의 공존

ZEC Toolkit은 네이밍과 구조를 분리하여 다른 플러그인과 충돌 없이 사용할 수 있습니다:

| 플러그인 | 역할 | 충돌 여부 |
|----------|------|-----------|
| oh-my-claudecode | 에이전트 실행 인프라 | 충돌 없음 |
| claude-code-harness | 스킬 원본 저장소 | Harness 스킬 중복 (ZEC 버전 우선) |
| 기타 커스텀 플러그인 | - | `zec-` 접두사로 분리 |

## 관리 명령어

| 작업 | 명령어 |
|------|--------|
| 설치 | `claude plugin install zec-toolkit@lemonbalms-zec-toolkit` |
| 업데이트 | `claude plugin update zec-toolkit@lemonbalms-zec-toolkit` |
| 제거 | `claude plugin uninstall zec-toolkit@lemonbalms-zec-toolkit` |
| 활성화 | `claude plugin enable zec-toolkit@lemonbalms-zec-toolkit` |
| 비활성화 | `claude plugin disable zec-toolkit@lemonbalms-zec-toolkit` |

<Callout type="warning">
제거 시 프로젝트의 `.zec/` 디렉토리는 삭제되지 않습니다. 설정을 완전히 제거하려면 수동으로 삭제하세요.
</Callout>

## 문제 해결

### 명령어를 찾을 수 없음

```
> /zec
Error: Unknown command
```

Claude Code에서 `/plugin` → **Installed** 탭에서 `zec-toolkit`이 있는지 확인합니다. 없다면 재설치합니다.

### 스킬이 로드되지 않음

Skills 디렉토리 경로가 `plugin.json`에 올바르게 설정되어 있는지 확인합니다:

```json
{
  "skills": "./skills/"
}
```
