# Harness 스킬 카탈로그

ZEC Toolkit에 포함된 9개 Harness 스킬입니다. 각 스킬은 특정 기술 스택에 대한 베스트 프랙티스와 패턴을 제공합니다.

## 백엔드

### harness-fastapi

FastAPI 백엔드 개발을 위한 DDD(Domain-Driven Design) 패턴 가이드입니다.

| 리소스 | 내용 |
|--------|------|
| `api-routes.md` | 라우터 설계 패턴 |
| `async-patterns.md` | async/await 패턴 |
| `database-orm.md` | SQLModel/SQLAlchemy ORM |
| `domain-driven-design.md` | DDD 원칙 |
| `dtos-validation.md` | Pydantic DTO + 유효성 검증 |
| `error-handling.md` | 에러 핸들링 |
| `layered-architecture.md` | 레이어드 아키텍처 |
| `repository-pattern.md` | Repository 패턴 |
| `service-layer.md` | Service 레이어 |
| `complete-examples.md` | 전체 예제 |

### harness-react-perf

Vercel Engineering 기반 React/Next.js 성능 최적화 **47개 룰**입니다.

카테고리별 룰:

| 카테고리 | 룰 수 | 예시 |
|----------|-------|------|
| Rendering | 9개 | content-visibility, hydration, svg-precision |
| Re-render | 12개 | memo, derived-state, transitions, lazy-state |
| JS | 10개 | Set/Map lookups, early-exit, cache, combine-iterations |
| Bundle | 5개 | dynamic-imports, barrel-imports, preload |
| Async | 5개 | parallel, suspense-boundaries, defer-await |
| Server | 6개 | cache-react, parallel-fetching, auth-actions |

### harness-frontend-design

프로덕션 수준의 프론트엔드 UI 디자인을 생성합니다. 일반적인 AI 미학을 피하고 창의적이고 세련된 코드를 생성합니다.

### harness-web-design

Web Interface Guidelines 기반 UI 코드 리뷰를 수행합니다. WCAG 접근성 준수를 검사합니다.

## 도구

### harness-mermaid

18가지 이상의 Mermaid 다이어그램을 생성합니다.

| 다이어그램 | 레퍼런스 |
|-----------|---------|
| Flowchart, Sequence, Class | 기본 다이어그램 |
| ER, State, Gantt | 설계 다이어그램 |
| Git Graph, Mindmap, Timeline | 시각화 |
| C4, Architecture | 아키텍처 |
| Sankey, Radar, XY Chart | 데이터 차트 |
| Kanban, Treemap, Packet | 특수 다이어그램 |

22개 레퍼런스 파일 + 설정(theming, layouts, directives) 포함.

### harness-pdf

PDF 파일 조작 도구입니다. 텍스트 추출, 병합, 분할, 워터마크, 폼 작성, OCR 등을 지원합니다.

### zec-tool-ast-grep

AST(Abstract Syntax Tree) 기반 구조적 코드 검색과 변환 도구입니다. 대규모 코드 변경, 보안 패턴 스캔, API 마이그레이션에 사용됩니다.
